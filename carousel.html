Module shinyant.carousel

Carousel component module for shinyant UI library.

This module provides an Ant Design-inspired carousel component with various configuration options for slideshow presentation.

Key features include:

  • Support for any content types in slides

  • Customizable navigation (arrows, dots)

  • Multiple dot positioning options (top, bottom, left, right)

  • Autoplay with configurable speed

  • Smooth transition animations

  • Responsive height adaptation

  • Draggable navigation on desktop

  • Infinite looping mode

  • Customizable design tokens

  • etc.

The component follows Ant Design specifications while adapting them for Python usage.

Example

from shiny import App, ui

import shinyant


content_style_str = "margin:0; height:160px; color:#fff; line-height:160px; text-align:center; background:#364d79; background:#364d79; font-family:'Open Sans';"

app_ui = ui.page_auto(
    shinyant.nesting_wrap(
        shinyant.carousel(
            id="carousel",
            items=[
                ui.h3("Slide 1 Content", style=content_style_str),
                ui.h3("Slide 2 Content", style=content_style_str),
                ui.h3("Slide 3 Content", style=content_style_str),
            ],
            fade=True,
            autoplay=True,
            dotPosition="left",
        )
    )
)


def server(input, output, session):
    pass


app = App(app_ui, server)

def carousel(
    id: str,
    items: list[Any] | None = None,
    arrows: bool = False,
    autoplay: bool | dict[str, bool] = False,
    autoplaySpeed: int = 3000,
    adaptiveHeight: bool = False,
    dotPosition: DotPositionTypes = "bottom",
    dots: bool = True,
    draggable: bool = False,
    fade: bool = False,
    infinite: bool = True,
    speed: int = 500,
    waitForAnimate: bool = False,
    design: CarouselDesignTokens | None = None,
    globalDesign: CarouselGlobalTokens | None = None,
):
    """
    Creates a carousel component based on ant design react Carousel component

    Attributes:
        id (str): name of the component in the application
        items (list[Any]): list of components to display on carousel slides. Defauls to None
        arrows (bool): Whether to show switch arrows. Defauls to False
        autoplay (bool | dict): Whether to scroll automatically, you can specify autoplay={'dotDuration': True} to display the progress bar. Defauls to False
        autoplaySpeed (int): Delay between each auto scroll (in milliseconds). Defauls to 3000
        adaptiveHeight (bool): Adjust the slide's height automatically. Defauls to False
        dotPosition (str): The position of the dots, which can be one of 'top', 'bottom', 'left', 'right'. Defauls to 'bottom'
        dots (bool): Whether to show the dots at the bottom of the gallery. Defauls to True
        draggable (bool): Enable scrollable via dragging on desktop. Defauls to False
        fade (bool): Whether to use fade transition. Defauls to False
        infinite (bool): Infinitely wrap around contents. Defauls to True
        speed (int): Animation speed in milliseconds. Defauls to 500
        waitForAnimate (bool): Whether to wait for the animation when switching. Defauls to CarouselDesignTokens
        design (CarouselDesignTokens): Design of carousel (CarouselDesignTokens class object). Defauls to CarouselDesignTokens
        globalDesign (CarouselGlobalTokens): Global design of carousel (CarouselGlobalTokens class object). Defauls to CarouselGlobalTokens
    """
    if design is None:
        design = CarouselDesignTokens()
    if globalDesign is None:
        globalDesign = CarouselGlobalTokens()

    return Carousel(
        id=id,
        items=items,
        arrows=arrows,
        autoplay=autoplay,
        autoplaySpeed=autoplaySpeed,
        adaptiveHeight=adaptiveHeight,
        dotPosition=dotPosition,
        dots=dots,
        draggable=draggable,
        fade=fade,
        infinite=infinite,
        speed=speed,
        waitForAnimate=waitForAnimate,
        design=design,
        globalDesign=globalDesign,
    )

    Attributes:
        colorBgContainer (str): Container background color, e.g: default button, input box, etc. Be sure not to confuse this with `colorBgElevated`
        colorText (str): Default text color which comply with W3C standards, and this color is also the darkest neutral color
        fontFamily (str): The font family of Ant Design prioritizes the default interface font of the system, and provides a set of alternative font libraries that are suitable for screen display to maintain the readability and readability of the font under different platforms and browsers, reflecting the friendly, stable and professional characteristics
        fontSize (int): The most widely used font size in the design system, from which the text gradient will be derived
        lineHeight (float): Line height of text
        marginXXS (int): Control the margin of an element, with the smallest size
        motionDurationSlow (str): Motion speed, slow speed. Used for large element animation interaction
    """
